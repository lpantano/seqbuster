What is needed?:

  • Java version 1.6 You can get it: http://java.sun.com/javase/downloads/index.jsp. If you are running in Mac OSX Leopard 10.5 you have to upgrade to Snow Leopard 10.6 to use the tool.

  • Blast+ repository 2.2.X or greater (megablast has been changed to blastx) You can get it: ftp://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/LATEST/. If you are a macOSX user, select the DMG file (ncbi-blast-2.2.23+.dmg). Also you will find a tutorial with help for installation.

  • Before installing MySQL, read this step if you are an administrator : If do not want that SeqBuster has the total permissions of all the database, we suggest to run other mysql server separatly using the steps explained here: http://code.openark.org/blog/mysql/manually-installing-multiple-mysql-instances-on-linux-howto
  • MySQL You can get it from http://dev.mysql.com/downloads/mysql/. Choose you system: Mac OSX (look for the "Mac OSX ver. 10.6 DMG" file) or Linux. Here you can register to MySQL forum or skip this step (there is a link smaller indicating that you only want to download MySQL).
  • For macOSX users: install first the mysql-XXX-osxXXX-XXX.pkg file (where X are numbers) and then the MYSQLStartupitem.pkg file. After that you should restart the computer.
  • Follow these instruction if you want to do more secure your connection,but it is not a mandatory step. So if you stop here, your user for mysql in the SeqBuster configuration will be root and the password will be empty.

    • Open a terminal(for macs ox users: it is in Aplication => tools folder)
    • Mac OS X
      • Type (change 'newpassword' to the password you want only with alpha-numeric characters): /usr/local/mysql/bin/mysqladmin -u root password newpassword
      • Type and press enter: /usr/local/mysql/bin/mysql -u root -p
      • You have to launch mysql every time you restart your computer, if you want to do automatically, install MySQLStartupItem.pkg that is in the dmg file
      • write password you have chosen in the previous step
    • Linux
      • Type (change 'newpassword' to the password you want only with alpha-numeric characters): mysqladmin -u root password newpassword
      • Type and press enter: mysql -u root -p
      • write password you have chosen in the previous step
    • Type (change write_yout_password for the password you want): CREATE USER 'seqbuster'@'localhost' IDENTIFIED BY 'write_yout_password';
    • Type: GRANT ALL PRIVILEGES ON *.* TO 'seqbuster'@'localhost' WITH GRANT OPTION;
    • exit
    At the end you will have an user: seqbuster and a password:write_yout_password that will be ask when you configure SeqBuster.
  • R statistic program You can get it from http://www.r-project.org/. Choose your country and then your system: mac OSX or Linux.







    Select the latest version (first link you see).
  • XML package needed to generate the output files:
    • Linux
      • Open a terminal and type su R and then type root password
      • type: install.packages('XML') and follow the instructions
    • Mac OSX
      • Open the Terminal App (for macs ox users: it is in Aplication => tools folder) and type:sudo R and type password (your user password used to log in to computer)
      • type: install.packages('XML') and follow the instructions, normally here you have to select a country to download the package
  • RMySQL package needed for the interaction between the analyses and the data storage:
    • type: install.packages('RMySQL') and follow the instructions
